# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/NightmareYeager/dns-switcher.git
```

Go into the project

```bash
cd dns-switcher
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python dns_switcher.py
```

# 🏎 Benchmark Engine

Unlike traditional ping tests, DNS Switcher performs **real DNS resolution benchmarking**.

Each server is tested using an actual DNS lookup to provide realistic performance measurements.
